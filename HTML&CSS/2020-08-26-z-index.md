# z-index 속성

## z-index의 정의

- CSS z-index 속성은 위치 지정 요소와, 그 자손 또는 하위 플렉스 아이템의 Z축 순서를 지정한다. 더 큰 z-index 값을 가진 요소가 작은 값의 요소 위를 덮는다.
- 포토샵, 일러스터 같은 걸로 비유하자면 레이어라고 생각하면 된다.
- 파워포인트에서 뒤로 보내기, 앞으로 가져오기와 같다고 생각하면 된다.
- z-index의 값이 클수록 앞에 오게 된다.
