- `configuration` : 배치, 배열

- `central` : 중앙의, 중요한
- `a number of trails` : 여러 가지 길로
- `underlying` : 근본적인
- `mechanisms` : 방법, 메커니즘

- `uniform` : 균일한, 한결같은

- `cater` : 공급하다
